<BR>

```math
c = \sqrt{a^2 + b^2}
```

# Distance Between Two Points

<BR>

```math
d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}
```

<BR>

# Midpoint Between Two Points

<BR>

```math
M = \left( \tfrac{x_1 + x_2}{2}, \tfrac{y_1 + y_2}{2} \right)
```
