# Area of a Rectangle

```math
A = \text{length} \times \text{width}
```

Create a function that takes the `length` and `width` of a rectangle and returns the rectangle's _area_.

Example:

```text
length = 3
width = 4
```

Expected Return Value:

```text
20
```
