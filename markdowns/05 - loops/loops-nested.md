# Exercise #1: Coordinate Grid

Given a number `n`. Print all (row, col) coordinate pairs in a grid from (0, 0) to (`n`, `n`).

__Example:__

`n` = 2

Expected Output:

```
(0,0) (0,1) (0,2)
(1,0) (1,1) (1,2)
(2,0) (2,1) (2,2)
```
