# Quiz: Count the Statements

# Question 1

```python
x = 5
y = 10
print(x + y)
```

?[How many statements?]
-[] 1
-[] 2
-[x] 3
-[] 4

<details>
<summary>Explanation</summary>

The correct answer is 3.

* `x = 5` → 1 statement
* `y = 10` → 1 statement
* `print(x + y)` → 1 statement

Total = 3 statements (imports don’t count, but there aren’t any here).

</details>
